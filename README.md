# Usage
<!-- usage -->
```sh-session
$ npm install -g pdfcheck
$ pdfcheck COMMAND
running command...
$ pdfcheck (-v|--version|version)
pdfcheck/0.0.1 darwin-x64 node-v12.13.1
$ pdfcheck --help [COMMAND]
USAGE
  $ pdfcheck COMMAND
...
```
